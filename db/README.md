# Start the service

`python csv_server.py`

# Query By Name (case insensitive)

`http://localhost:8080/by-name?name=bruno%20fernandes`

# Query By Row Number

`http://localhost:8080/by-row?num=123`

# Return Data Format

`name, team, number, nation, position("pos1,pos2,..."), age(years-days), minutes played, goals, assists`
